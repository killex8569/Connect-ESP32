# Connect-ESP32

Projet de connexion de deux esp 32 via protocole ESP Now en C.

## Objectif

En utilisant ESP Now, nous allons pouvoirs connecter deux ESP 32 entre eux. L'idée c'est de faire en sorte d'utiliser l'ESP 32.

## Structure du projet 

Fichier main.cpp  --> Va pouvoirs lancer l'emetteur et le recepteur. Choix entre les deux, on choisit soit l'emetteur soit le recepeteur, puis une carte peux soit être emettrice, soit receptrice.

On implémente une sécurité, avec surement un port, un mots de passe etc... pour le moment en claire dans le code puis on verra après.
